例子 1：資料怎麼看

這個 demo 把元素資料直接畫到固定的邏輯畫布上。重點不是螢幕尺寸，而是 `left / top / width / height` 在 `1000 x 562.5` 這個世界裡的意思。

元素資料

先把它當成投影片世界裡的矩形定義，不要先把它理解成螢幕像素大小。

const element = { left: 100, top: 50, width: 200, height: 80, }
  • left: 100
    距離邏輯畫布左邊 100。
  • top: 50
    距離邏輯畫布上方 50。
  • width: 200
    邏輯寬度是 200。
  • height: 80
    邏輯高度是 80。
邏輯畫布：1000 x 562.5
logic box

文字框

它站在邏輯座標 (100, 50)，尺寸是 200 x 80。
