例子 3：滑鼠為什麼要除以 canvasScale

畫面上的滑鼠點是螢幕座標，但投影片資料要寫回邏輯座標。請在右側畫布點一下， 你會同時看到 `screen` 與 `world` 兩組數值，並對照 `world = screen / canvasScale`。

目前倍率 0.5
最近一次點擊的螢幕座標 screenX: -, screenY: -
換算後的邏輯座標 worldX: -, worldY: -
const worldX = screenX / canvasScale const worldY = screenY / canvasScale

點擊縮放後的畫布

同一筆元素資料放在邏輯畫布 `(100, 50)`。畫面上的位置會隨倍率改變，但資料定義不變。

文字框

邏輯座標固定是 left 100 / top 50

如果倍率是 `0.5`，你在畫面上點到 `screenX = 50`，它其實對應的是 `worldX = 100`。因為這張畫布只是被整體縮小，邏輯世界本身沒有變小。
