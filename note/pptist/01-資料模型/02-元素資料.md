# 元素資料

`PPTElement` 是投影片上所有元素的總型別。

對應原始碼：

- `PPTist-SourceCode/src/types/slides.ts`
- `PPTist-SourceCode/src/views/components/element`
- `PPTist-SourceCode/src/configs/element.ts`
- `PPTist-SourceCode/src/configs/shapes.ts`
- `PPTist-SourceCode/src/configs/lines.ts`
- `PPTist-SourceCode/src/configs/chart.ts`
- `PPTist-SourceCode/src/configs/latex.ts`
- `PPTist-SourceCode/src/configs/imageClip.ts`
- `PPTist-SourceCode/src/utils/element.ts`

## 共用基底欄位

幾乎所有元素都有這些欄位：

- `id`
- `left`
- `top`
- `width`
- `height`
- `rotate`
- `lock?`
- `groupId?`
- `link?`
- `name?`

### 基底欄位補充

- `link?`
  - 型別是 `PPTElementLink`
  - 結構是 `{ type: 'web' | 'slide', target: string }`
- `groupId?`
  - 表示這個元素屬於哪個群組
- `lock?`
  - 表示元素是否鎖定

## `PPTElement` 的子型別

`PPTElement` 是以下子型別的 union：

- `text`
- `image`
- `shape`
- `line`
- `chart`
- `table`
- `latex`
- `video`
- `audio`

下面逐一對照實際欄位。

## 文字元素

### `PPTTextElement`

欄位：

- `type: 'text'`
- `content`
- `defaultFontName`
- `defaultColor`
- `outline?`
- `fill?`
- `lineHeight?`
- `wordSpace?`
- `opacity?`
- `shadow?`
- `paragraphSpace?`
- `vertical?`
- `textType?`

### `TextType`

`textType` 的值不是任意字串，而是：

- `title`
- `subtitle`
- `content`
- `item`
- `itemTitle`
- `notes`
- `header`
- `footer`
- `partNumber`
- `itemNumber`

## 圖片元素

### `PPTImageElement`

欄位：

- `type: 'image'`
- `fixedRatio`
- `src`
- `outline?`
- `filters?`
- `clip?`
- `flipH?`
- `flipV?`
- `shadow?`
- `radius?`
- `colorMask?`
- `imageType?`

### `ImageType`

`imageType` 的值是：

- `pageFigure`
- `itemFigure`
- `background`

### 其他圖片相關型別

- `filters` 對應 `ImageElementFilters`
- `clip` 對應 `ImageElementClip`
- `clip.range` 的型別是 `[[number, number], [number, number]]`

## 圖形元素

### `PPTShapeElement`

欄位：

- `type: 'shape'`
- `viewBox`
- `path`
- `fixedRatio`
- `fill`
- `gradient?`
- `pattern?`
- `outline?`
- `opacity?`
- `flipH?`
- `flipV?`
- `shadow?`
- `special?`
- `text?`
- `pathFormula?`
- `keypoints?`

### `ShapeText`

如果圖形內有文字，`text` 的結構是：

- `content`
- `defaultFontName`
- `defaultColor`
- `align`
- `type?`

`align` 的值是：

- `top`
- `middle`
- `bottom`

## 線條元素

### `PPTLineElement`

這個型別和 base element 不完全一樣，它是：

- `Omit<PPTBaseElement, 'height' | 'rotate'>`

欄位：

- `type: 'line'`
- `start`
- `end`
- `style`
- `color`
- `points`
- `shadow?`
- `broken?`
- `broken2?`
- `curve?`
- `cubic?`

### 補充

- 線條沒有 `height`
- 線條沒有 `rotate`
- `points` 是 `[LinePoint, LinePoint]`
- `LinePoint` 的值是：
  - `''`
  - `arrow`
  - `dot`

## 圖表元素

### `PPTChartElement`

欄位：

- `type: 'chart'`
- `fill?`
- `chartType`
- `data`
- `options?`
- `outline?`
- `themeColors`
- `textColor?`
- `lineColor?`

### `ChartType`

值有：

- `bar`
- `column`
- `line`
- `pie`
- `ring`
- `area`
- `radar`
- `scatter`

## 表格元素

### `PPTTableElement`

  - `color?`
  - `backcolor?`

  - `fontname?`

## 小結

如果只看外層筆記，很容易把 `PPTElement` 當成「很多欄位拼在一起」。

實際上比較準確的理解方式是：

- 先看共用 base 欄位
- 再看每個 `type` 對應的子型別
- 再看各自的巢狀型別與 union 值

這樣會更接近 `pptist` 原始碼，也更方便你後續看元件實作。
